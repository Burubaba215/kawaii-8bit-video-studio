# 🎀 KAWAII 8BIT VIDEO STUDIO v3

レトロ × 16色パレット × 8bit演出  
ブラウザだけで動く、PC向けミニ動画スタジオ。

---

## ▶ Web版（GitHub Pages）

🔗 https://burubaba215.github.io/kawaii-8bit-video-studio/

※ PC専用（マウス操作前提）

---

## 🎬 できること

- MP4 / MP3 / PNG の読み込み
- タイムライン編集
- 16色パレット字幕
- アウトライン / 影 / BGボックス
- 8bit風エフェクト（Scanline / RGB Shift など）
- WebM書き出し（YouTube / ニコニコ対応）

---

## 🖱 使い方

1. **IMPORT FILE** で素材を読み込み
2. タイムラインにドラッグ
3. 字幕を入力・スタイル調整
4. ▶ PLAY で確認
5. EXPORT でWebM出力

---

## ⚠ 注意

- PCブラウザ専用
- 書き出し形式は WebM
- データ保存はブラウザローカル
